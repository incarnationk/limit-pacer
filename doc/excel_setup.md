# Excel データソース作成手順 (Excel Setup Guide)

SharePoint の代わりに **個人の OneDrive 上の Excel ファイル** をデータベースとして使用します。

## 1. Excelファイルの作成
1. [OneDrive](https://onedrive.live.com/) (または Microsoft 365 ポータル) を開きます。
2. 「自分のファイル (My files)」の直下（ルートフォルダ）に、新規 Excel ブックを作成します。
3. ファイル名を **`limit-pacer.xlsx`** に変更します。

---

## 2. シートとテーブルの作成
このファイルの中に2つのシートを作成し、それぞれに「テーブル」を設定します。
**※注意: テーブル名は正確に設定してください。**

### シート1: `Members`
1. シート名を `Members` に変更します。
2. 1行目に以下のヘッダーを入力します：
   `id`, `name`, `team`, `role`, `email`
3. データ（例: 自分自身の情報など）を数行入力します。
4. 入力範囲を選択し、**「挿入」→「テーブル」** をクリックしてテーブル化します。
5. **「テーブルデザイン」** タブで、テーブル名を **`Start_Members`** に変更します。
     - ※デフォルトの `テーブル1` などから必ず変更してください！

**データ例:**
| id | name | team | role | email |
|---|---|---|---|---|
| 1 | 山田 太郎 | 開発 | リーダー | (自分のMSアカウントのアドレス) |
| 2 | 鈴木 一郎 | 営業 | 社員 | |

---

### シート2: `Tasks`
1. 新しいシートを追加し、シート名を `Tasks` に変更します。
2. 1行目に以下のヘッダーを入力します：
   `id`, `content`, `deadline`, `target`, `link`, `completedBy`
3. テストデータを数行入力します。
4. 範囲を選択し、**「挿入」→「テーブル」** をクリックしてテーブル化します。
5. **「テーブルデザイン」** タブで、テーブル名を **`Start_Tasks`** に変更します。

**データ例:**
| id | content | deadline | target | link | completedBy |
|---|---|---|---|---|---|
| 101 | 年末調整 | 2025-12-10 | 全メンバー | https://example.com | |
| 102 | 目標設定 | 2025-04-01 | 社員 | https://example.com | |

---

## 3. 完了
これで準備完了です。
ファイル名 `limit-pacer.xlsx` と、テーブル名 `Start_Members`, `Start_Tasks` が正しければ、アプリから読み込めるようになります。
